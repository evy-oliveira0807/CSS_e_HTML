FlexBox

O CSS Flexbox é um modelo de layout que permite o design de um layout flexível e eficiente. É útil para criar layouts responsivos.

Inline Block

Elementos com display "inline-block" ficam em linha, mas permitem ajustes de largura e altura. Ótimo para criar blocos com conteúdo.
